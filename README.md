Pathfinderz Travel 🌍✈️

A responsive and modern travel website built using only HTML and CSS.  
Pathfinderz Travel showcases beautiful travel destinations with a clean and user-friendly interface.

🚀 Features

- Responsive Design
- Modern Travel Website UI
- Home Page
- About Page
- Contact Page
- Clean Layout
- Beginner-Friendly Project
- Pure HTML & CSS

🛠️ Technologies Used:

- HTML5
- CSS3

🎯 Project Purpose:
This project was created to practice front-end web development skills and improve understanding of responsive website design using HTML and CSS.
